# **Main pattern from this batch**

The biggest cleanup is this:

Role-specific starters should usually be Tactical Starter,  
not Heavy-Minute Starter.

Examples:

Duncan Robinson  
Joe Harris  
Seth Curry  
Reggie Bullock  
Robert Covington  
Jae Crowder  
Justin Holiday

These guys matter a lot, but their team role is narrow. That distinction helps shrinkage because they should not get the same responsibility prior as a true high-burden starter.

Next batch should be **2022-23 and 2023-24**, where the leaguewide cleanup gets more interesting because a lot of young players jump roles and several older stars stop being true cornerstone/starter-status players.
